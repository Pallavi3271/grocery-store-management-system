# Grocery Store Management System

This project is a simple web application to manage grocery products and billing.

## Features
- Add, update, and delete products  
- Add items to cart and calculate total  
- Automatically update stock after purchase  
- Generate invoice in PDF format  

## Technologies Used
- HTML, CSS, JavaScript  
- Python (Flask)  
- REST API  

## How to Run

1. Go to backend folder  
   python server.py  

2. Open frontend/index.html in browser  

## About

This project helped me understand how frontend and backend work together and how to build a simple full-stack application.
